# Gravit'around

A web app to watch your favourite satellites in real time around Earth.

Prerequisites
------------

You'll need :

    - Docker

Installation
------------

Tap in your terminal ::

    $ git clone https://github.com/thomassimmer/gravitaround.git
    $ cd gravitaround
    $ docker-compose build
    $ docker-compose up
